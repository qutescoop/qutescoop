QuteScoop

Help, information and bug reporting

Find more information and help in the QuteScoop Documentation [https://qutescoop.github.io/qutescoop/].
The most active QuteScoop-users forum is on VATSIM Germany: http://board.vacc-sag.org/49/28031 [german, but use any language you like]
If you want to report a bug, read the Wiki for how to obtain a 'debug log' and attach it to your bug report, please!!!.
Report bugs to our bugtracker: [https://github.com/qutescoop/qutescoop/issues]
